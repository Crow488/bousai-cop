# 防災COP — 佐世保 状況認識ダッシュボード

地域の防災情報（気象警報・雨雲・危険度分布・地震・避難所）を**1画面に統合**した市民向けの状況認識（Situational Awareness）ツール。

## コンセプト — 軍事的状況認識の民生転用



本ツールはその発想を防災に転用したもの：

- **情報統合**: 気象庁・国土地理院に分散する情報源を1画面に集約
- **状況判断の自動化**: 警報・地震情報を統合し、総合状況レベル（平常/注意/警戒/危険）を根拠つきで提示
- **情報の鮮度管理**: 全パネルに更新時刻と鮮度インジケータを表示（古い情報で判断しないための軍隊的習慣）
- **DTG表示**: 日時群（陸自・米軍式日付時刻表記）を併記

## 機能

| パネル | 内容 | ソース | 自動更新 |
|---|---|---|---|
| 地図（COP主画面） | 避難所435件・雨雲ナウキャスト・キキクル（洪水/土砂/浸水）・地震震央・行政界 | 地理院タイル + 気象庁タイル | 5分 |
| 状況判断 | 警報＋近傍地震（150km/24h以内）を統合した総合レベル | 自動演算 | 連動 |
| 警報・注意報 | 佐世保市・佐世保市（宇久）・佐々町の発表状況（気象庁公式配色） | 気象庁 | 5分 |
| 気象実況 | アメダス佐世保・平戸・松浦（気温/雨量1h・24h/風/湿度） | 気象庁 | 10分 |
| 天気予報 | 北部の今日明日＋降水確率＋佐世保の気温 | 気象庁 | 30分 |
| 地震 | 直近の地震（近傍はハイライト、震央を地図にプロット） | 気象庁 | 5分 |
| 避難所検索 | 施設名・住所・災害種別で絞り込み → クリックで地図移動 | 国土地理院 | 静的 |

## アーキテクチャ

```
ブラウザ（静的HTML + vanilla JS + Leaflet 1.9.4）
 ├─ 気象庁 bosai API（CORS開放済み・キー不要）… 警報/予報/アメダス/地震
 ├─ 気象庁 jmatile … 雨雲ナウキャスト・キキクルのPNGタイル
 ├─ 地理院タイル（淡色地図）
 └─ ローカル静的データ data/*.geojson … 避難所・行政界（出典: data/SOURCES.md）
```

サーバー不要・APIキー不要・ビルド不要。`python3 -m http.server` などで配信するだけで動く。

## 実行方法

```bash
cd （リポジトリのルート）/
python3 -m http.server 8745
# → http://localhost:8745
```

## 別の地域への移植

`js/config.js` の予報区コード・市町村コード・アメダス観測点・地図中心を書き換えるだけ。コードは
https://www.jma.go.jp/bosai/common/const/area.json で調べられる。

## 出典・ライセンス

- 気象データ・危険度分布・雨雲: 気象庁
- 避難所データ: 国土地理院「指定緊急避難場所データ」（詳細は `data/SOURCES.md`）
- 地図タイル: 国土地理院（地理院タイル）
- 行政界: 「国土数値情報（行政区域データ）」（国土交通省）を加工して作成（smartnews-smri/japan-topography 経由）
- 地図ライブラリ: [Leaflet](https://leafletjs.com/) v1.9.4（BSD-2-Clause、GitHub公式リリースを同梱）

## 免責

本サイトは参考情報です。**避難の判断は市町の避難情報・気象庁の発表に従ってください。**
避難所データは最新でない場合があります。最新の状況は市町にご確認ください。
